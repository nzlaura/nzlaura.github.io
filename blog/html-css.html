Borders

To start off with, we can talk about the border. The border is something you can add around your content, for example adding a border to an image. Borders can be solid, like this code that I have written and applied to one image. This gives our image a thick, solid, purple border.

Margins

Next, margins! Margin is a way of describing the space around the outside of your content. You can use the margin to control the space around the element that you've created. For example, you can use "margin-bottom: 20px;" to put a small amount of space betwen the content you put in your box, and the content below it. Padding

Padding can be described as the space between the content and the border of the box. We can use padding like this, to move our image to the right of the "box", or the webpage that we are working on.
